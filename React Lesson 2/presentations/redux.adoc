== Core Concepts

* State: Appâ€™s state is described as a plain object
* Action: An action is a plain JavaScript object that describes what happened
* Reducer: To tie state and actions together. Function that takes state and action as arguments, and returns the next state

== Key Concepts: Store

.Create a store
[source,javascript,indent=0]
----
const todoApp = combineReducers({
  todos,
  visibilityFilter
});

const store = createStore(todoApp);
----

== Key Concepts: Dispatch

.Dispatch an action
[source,javascript,indent=0]
----
store.dispatch({
  type: 'ADD_TODO',
  id: 1,
  text: 'Go shopping'
});
----

== Key Concepts: Action Creator

.Setup State
[source,javascript,indent=0]
----
const addTodo = (id, text) => {
  return {
    type: 'ADD_TODO',
    id, // remember ES6 object short notation
    text,
  };
}
----

== Key Concepts: Reducer

.Setup State
[source,javascript,indent=0]
----
function todoApp(state = initialState, action) {
  switch (action.type) {
    case ADD_TODO:
      return [
        ...state,
        {
          text: action.text,
          completed: false
        }
      ]
    default:
      return state
  }
}
----
